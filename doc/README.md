## 使用指南

1. 按照需求查看对应章节
2. 开发新功能前建议先查阅架构概览和代码规范
3. 为已有模块添加功能时参考对应模块文档
4. 添加新API时参考API接口文档格式

## 文档维护

本文档应随代码更新而更新，特别是在以下情况：

- 添加新的模块或服务

- 更新技术栈或依赖
